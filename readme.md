# PICOBOY

A Mega Man-like code exploration adventure in Pico 8.

## Controls

⬆️ Climb up ladders  
⬇️ Climb down ladders / Combine with 🅾️ to drop through semisolids  
⬅️ Move left  
➡️ Move right

🅾️ Jump / Let go of ladders  
❎ Shoot
